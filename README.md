# KING STOR – Global Luxury Store

A bilingual (French/Arabic) e-commerce platform for luxury products with a modern, elegant design.

## ✨ Features

- 🌍 **Bilingual Support**: Seamlessly switch between French and Arabic
- 🛒 **Shopping Cart**: Add products to cart and manage orders
- 📱 **Responsive Design**: Works perfectly on desktop and mobile devices
- ⚙️ **Admin Panel**: Add and manage products with ease
- 💬 **WhatsApp Integration**: Send orders directly via WhatsApp
- 🎨 **Luxury Design**: Gold and black premium aesthetic

## 🚀 Live Demo

## 📋 How to Use

### For Customers:
1. Browse the product catalog
2. Click on a product to view details
3. Add items to your cart
4. Click "Send Order" to submit via WhatsApp

### For Admin:
1. Scroll to the bottom right (⚙ Admin panel)
2. Enter password: `1234`
3. Click "Login"
4. Fill in product details and click "Add Product"
